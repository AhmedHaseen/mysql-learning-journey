# MySQL Learning Journey

This repository documents my structured learning journey in MySQL, following a progressive path from beginner to advanced concepts. Each topic includes hands-on practice queries to strengthen understanding and real-world application.

---

## 📌 Overview

The purpose of this repository is to:

- Build a strong foundation in MySQL
- Practice writing efficient SQL queries
- Understand data manipulation and analysis
- Track consistency and learning progress over time

---

## 📚 Topics Covered

### 🔹 Beginner Level

- SELECT statements
- Filtering (WHERE, ORDER BY)
- LIMIT clause
- Aliasing

---

### 🔹 Intermediate Level

- Joins (INNER, LEFT, RIGHT)
- Group By & Aggregations
- String Functions
- CASE Statements
- Subqueries
- Window Functions

---

### 🔹 Advanced Level

- Common Table Expressions (CTEs)
- Temporary Tables
- Stored Procedures
- Triggers & Events

---

## 🧠 Learning Approach

- Practiced each concept with **hands-on SQL queries**
- Focused on **understanding logic and use cases**
- Followed a **step-by-step progression** from basics to advanced topics
- Improved queries through continuous practice and refinement
